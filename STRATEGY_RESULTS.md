## 🏆 최종 추천 전략: RSI Divergence

### 전략 설명
가격이 신저가를 갱신하는데 RSI는 상승할 때 (상승 다이버전스) 롱 진입
가격이 신고가를 갱신하는데 RSI는 하락할 때 (하락 다이버전스) 숏 진입

### 최적 파라미터
```python
# 대상 코인 (4개만!)
COINS = ['BTCUSDT', 'ETHUSDT', 'BNBUSDT', 'HYPEUSDT']

# 전략 설정
LOOKBACK = 10           # 신고가/신저가 탐색 기간
DIV_THRESHOLD = 3       # RSI 최소 변화량
RSI_PERIOD = 14         # RSI 기간

# 손익비
ATR_SL = 0.7           # 손절: ATR × 0.7
ATR_TP = 2.0           # 익절: ATR × 2.0

# 레버리지/포지션 (MDD 40% 이하 최적)
LEVERAGE = 10
POSITION_PCT = 0.12    # 12%
```

## ⚠️ 핵심 교훈

1. **코인 수 늘리면 안됨**
   - 4코인: +89%, 30코인: -79%
   - 주요 코인에 집중해야 함

2. **레버리지 높이면 MDD 급증**
   - L10x: MDD 31%
   - L20x: MDD 64%

3. **손익비가 핵심**
   - ATR 0.7:2.0 (약 1:3 손익비) 최적
   - 손절 타이트하게, 익절은 여유롭게

4. **하루 10% 목표는 비현실적**
   - 일평균 최고 0.53%
